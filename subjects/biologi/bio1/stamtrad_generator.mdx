## Instruktioner för att spara bilder:

### Metod 1: Screenshot (enklast)
1. Öppna denna fil i din lokala dev-server
2. Ta screenshot av varje stamträd (Cmd+Shift+4 på Mac)
3. Spara i `/prov/subjects/biologi/bio1/images/`

### Metod 2: Browser DevTools (bättre kvalitet)
1. Högerklicka på stamträdet
2. "Inspect Element"
3. Högerklicka på `.pedigree-container` i DevTools
4. "Capture node screenshot"
5. Spara bilden

### Metod 3: Puppeteer/Playwright (automatiskt)
Om du vill automatisera kan vi skapa ett script som tar screenshots automatiskt.
